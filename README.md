# Project-Configurable-RV-
It consists of a set of functional blocks that enable the creation of scalar RV processors of various configurations with out-of-order instruction execution, as well as the integration of identical scalar RV processors into a single superscalar processor

The set includes:
Instruction fetch unit
Decoding unit
Data selection units and storage units for integer and floating-point instructions (reservation stations)
Execution units: ALUs for integer and floating-point operations, a data memory access unit, and a control transfer execution unit
Data bus
